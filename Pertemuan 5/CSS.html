Contoh Penggunaan Properti CSS

Properti text-indent

Ini adalah contoh penggunaan properti CSS. Properti text-indent akan menjorokkan teks sebesar 20 piksel dari batas kiri. Ini berguna untuk membuat paragraf terstruktur dengan baik.

Properti letter-spacing

Contoh ini menunjukkan penggunaan properti CSS letter-spacing yang mengatur spasi antara huruf-huruf menjadi lebih longgar.

Properti word-spacing

Contoh ini menunjukkan penggunaan properti CSS word-spacing untuk menambahkan spasi antara kata-kata dalam sebuah teks.

Properti line-height

Contoh ini menunjukkan penggunaan properti CSS line-height untuk mengatur tinggi baris dalam sebuah elemen teks. Ini dapat meningkatkan keterbacaan teks.

Properti white-space

Contoh ini menunjukkan penggunaan properti CSS white-space untuk mengontrol bagaimana spasi putih dalam teks ditangani.
